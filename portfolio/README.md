## 🚀 How to Use

### Option 1: Direct File Download
1. Download all three files (`index.html`, `styles.css`, `script.js`)
2. Place them in a single folder
3. Open `index.html` in your web browser

### Option 2: GitHub Pages Deployment
1. Create a new GitHub repository (e.g., `portfolio` or `joshua-stadler-portfolio`)
2. Upload all three files to the repository
3. Enable GitHub Pages in repository settings
4. Select "main" branch as source
5. Your site will be live at: `https://yourusername.github.io/repository-name`

### Option 3: Local Web Server
```bash
# Navigate to your project directory
cd /path/to/portfolio

# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js (if installed)
npx http-server
```

Then visit `http://localhost:8000` in your browser.

## 🎨 Customization Guide

### Colors
Edit CSS custom properties in `styles.css`:
```css
:root {
    --primary-color: #0066cc;      /* Main blue */
    --secondary-color: #00aa66;    /* Green accent */
    --accent-color: #ff6b35;       /* Orange accent */
    /* ... more colors */
}
```

### Content Updates
Edit sections in `index.html`:
- **Hero Title & Tagline**: Lines 39-42
- **About Cards**: Lines 57-80
- **Nursing Experience**: Lines 107-152
- **Professional Experience**: Lines 159-227
- **Contact Information**: Lines 243-264

### Typography
Modify font settings in `styles.css`:
```css
body {
    font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
    font-size: 1rem;
    line-height: 1.6;
}
```
